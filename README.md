# OpsFlow

OpsFlow is a full-stack incident ticket triage and SLA management system.

## Overview

OpsFlow helps support teams manage incidents from intake through resolution. Users can submit tickets, analysts can triage and work assigned tickets, and managers can monitor SLA risk, workload, and operational reporting.

## Planned Features

- Ticket submission
- Role-based access
- Priority and SLA calculation
- Assignment workflow
- Ticket comments
- Audit trail
- Analyst queue
- Manager dashboard
- CSV reporting
- Rule-based triage suggestions
- Automated testing
- CI/CD
- Cloud deployment

## Tech Stack

- Backend: Java, Spring Boot, Spring Security, Spring Data JPA
- Frontend: React, TypeScript, Vite
- Database: PostgreSQL
- Testing: JUnit, Spring Boot tests, frontend tests, end-to-end tests
- DevOps: Docker, GitHub Actions
